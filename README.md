# Flutter Week 1 Projects

This repository contains the Flutter projects completed during **Week 1** of my Flutter Development Internship. These projects focus on learning the fundamentals of Flutter, Dart, widget-based UI design, and state management.

## Projects Included

### 1. I Am Rich
A simple Flutter application demonstrating the basic structure of a Flutter app, including the use of MaterialApp, Scaffold, AppBar, Image, and Text widgets.

### 2. Mi Card
A personal profile card application showcasing Flutter UI design using widgets such as Column, Row, CircleAvatar, Card, and ListTile.

### 3. Dicee
An interactive Flutter application that simulates rolling two dice using StatefulWidget and setState() for dynamic UI updates.

## Skills Practiced

- Dart Programming
- Flutter Widgets
- StatelessWidget
- StatefulWidget
- Material Design
- UI Layouts
- State Management using setState()
- Asset Management
- Responsive UI Basics

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/prasanna0834/flutter-week1-projects.git
```

2. Open any project folder.

```bash
cd dicee_new
```

3. Install dependencies.

```bash
flutter pub get
```

4. Run the application.

```bash
flutter run
```
